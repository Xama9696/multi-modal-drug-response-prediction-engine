# Multi-Modal Drug Response Prediction Engine

A machine learning + deep learning system to predict drug candidate response using molecular and biological features.

## Objective
Classify candidates into:
- High Response
- Medium Response
- Low Response

## Models Used
- Random Forest Classifier
- Neural Network (TensorFlow / Keras)

## Features
- Molecular Stability
- Binding Affinity
- Solubility
- Toxicity
- Cell Response
- Dose Efficiency

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, TensorFlow, Matplotlib

## Evaluation
- Accuracy
- Classification Report
- Validation Curves
- Feature Importance

## Business Impact
Supports faster pharmaceutical experimentation and candidate prioritization.
